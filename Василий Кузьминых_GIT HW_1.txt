4) Создать внешний репозиторий c названием JSON - https://github.com/SkyXinergy/JSON
5) Клонировать репозиторий JSON на локальный компьютер - git clone https://github.com/SkyXinergy/JSON.git
6) Внутри локального JSON создать файл “new.json” - touch new.json
7) Добавить файл под гит - git add
8) Закоммитить файл - git commit -m "Created new json-file"
9) Отправить файл на внешний GitHub репозиторий - git push
10) Отредактировать содержание файла “new.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON - vim new.json, нажать i, ввести информацию, нажать ESC, нажать :, ввести wq и нажать ENTER 
11) Отправить изменения на внешний репозиторий. - git add, git commit -m "changes", git push
12) Создать файл preferences.json - touch preferences.json
13) В файл preferences.json добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате JSON. - vim preferences.json, нажать i, ввести информацию, нажать ESC, нажать :, ввести wq и нажать ENTER 
14) Создать файл sklls.json добавить информацию о скиллах которые будут изучены на курсе в формате JSON - cat >> skills.json, прописать информацию, нажать CTRL + C
15) Отправить сразу 2 файла на внешний репозиторий - git add, git commit -m "changes", git push
16-19) В веб-интерфейсе
20) git pull
